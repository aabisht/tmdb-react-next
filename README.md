This is a [Next.js](https://nextjs.org/) project using [`The Movie Database (TMDB)`](https://developer.themoviedb.org/docs).

## Commands

#### Command to install required package

```bash
npm install
```

#### Command to run server locally

```bash
npm run dev
```

#### Command to build the project

```bash
npm run build
```

#### Command to check lint errors

```bash
npm run lint
```

#### Command to format the code

```bash
npm run format
```

#### Command to run test case

```bash
npm run test
```
